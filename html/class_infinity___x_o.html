Detailed Description

3x3 board where pieces can be undone and oldest moves removed every 3 moves.

Implements the game logic for Infinity Tic-Tac-Toe (3x3 with move expiration).

The game is played on a standard 3x3 board, but after 9 moves, every new move replaces the oldest piece placed on the board.
