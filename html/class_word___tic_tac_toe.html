Detailed Description

3x3 letter-placement board which wins if any full line forms a valid dictionary word.

Implements the game logic for Word TicTacToe on a 3x3 board.

Players win by forming a valid 3-letter word horizontally, vertically, or diagonally. The game uses an external dictionary file for validation. Inherits from Board<char>.
