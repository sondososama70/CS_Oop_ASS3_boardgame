Detailed Description

Implements the board logic for Ultimate Tic-Tac-Toe.

This class manages the 3x3 grid of 3x3 sub-boards and tracks the winner of each sub-board on a separate main board. It enforces the rule that the location of the previous player's move dictates the next sub-board where the current player must play.
