Detailed Description

5x5 board where winner is determined by comparing 3-in-a-row counts.

Represents the 5x5 Tic-Tac-Toe game board where the winner is determined by counting 3-in-a-row sequences.

Attributes inherited: rows, columns, board, n_moves, blank_symbol.

The game is played on a 5x5 board, and the winner is the player who has more 3-in-a-row sequences (horizontal, vertical, or diagonal) when the board is full (24 moves). Inherits from Board<char>.
